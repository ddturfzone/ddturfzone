📊 Google Sheet Backup
Writes into your existing spreadsheet — open it here — never a new one. Slot Bookings, each Coaching sport, and Café Sales each get their own auto-created monthly sheet (e.g. "Slot Bookings - Jul 2026"), so nothing is ever mixed together.
Stops random people who guess your Apps Script URL from writing into your sheet. Set the same value in Code.gs's SHARED_SECRET constant.
Every booking create/edit/cancel, coaching fee payment, and café sale auto-posts here once set. Need the Apps Script code? It's the separate DDTZ_Apps_Script_Code.gs file provided alongside this app — paste it into Extensions → Apps Script on that spreadsheet and deploy as a Web App to get this URL, then use Test Connection above to confirm it's working before relying on it.
📥 Import Data
Migrate existing records from Excel (.xlsx) or CSV into DD Turf Zone. Every imported record is validated, checked against what's already here to skip duplicates, saved locally, and then automatically synced to Google Sheets — same as a record created directly in the app.
💾 Backup Settings
Manual on-demand Excel backups are available from Bookings and Reports. Browsers can't write files to disk automatically for security reasons — this reminder is the closest safe equivalent to "automatic" backups.
